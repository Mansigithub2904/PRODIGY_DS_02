# Task-02: Titanic Data Analysis

## Objective
Perform data cleaning and EDA to identify patterns.

## Features
- Data Cleaning & Feature Engineering
- Advanced EDA with Seaborn
- Correlation Heatmap
- Interactive Streamlit Dashboard

## Key Insights
- Females had higher survival rate
- Higher class → higher survival
- Fare positively correlated with survival

## Run Project

1. Install dependencies:
pip install -r requirements.txt

2. Run cleaning:
python scripts/data_cleaning.py

3. Run EDA:
python scripts/eda.py

4. Run dashboard:
streamlit run app/app.py
